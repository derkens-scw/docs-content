## Usage limits

This section contains usage limits that apply when using Messaging and Queuing.

**NATS**

| Resource                                       | Criteria   | Limits   | Scope        |
|------------------------------------------------|------------|----------|--------------|
| NATS accounts                                  | Max number | 10       | Organization |
| NATS message payload                           | Max size   | 512 KB   | Message      |
| NATS queue/stream storage                      | Max size   | 300 MB** | NATS account |
| Number of NATS streams                         | Max number | 10       | NATS account |
| Number of NATS consumers                       | Max number | 50       | NATS account |

**This implies stream replication. If you choose to replicate your stream across all nodes (3), a storage limit of 100 MB applies.

**SQS/SNS**

| Resource                                       | Criteria   | Limits   | Scope            |
|------------------------------------------------|------------|----------|------------------|
| SQS queue storage                              | Max size   | 100 MB   | Scaleway Project |
| Number of SQS queues                           | Max number | 15       | Scaleway Project |
| Number of SNS subscribers per topic            | Max number | 50       | Scaleway Project |
| Number of SNS topics                           | Max number | 15 days  | Scaleway Project |
| SQS/SNS message payload                        | Max size   | 256 KB   | Message          |
| Message retention period (SQS)                 | Max period | 14 days  | Message          |
| Message retention period (SQS)                 | Max period | 14 days  | Message          |
| Visibility timeout (SQS)                       | Max period | 12 hours | SQS queue        |


These limits are enforced as [Organization quotas](/console/my-account/reference-content/organization-quotas/).

## VPC

Messaging and Queuing is not currently compatible with [Scaleway VPC](https://www.scaleway.com/en/docs/network/vpc/quickstart/).
